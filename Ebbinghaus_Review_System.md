# 艾宾浩斯复习系统实现

## ? 核心功能

### 1. 时间戳记录
- 每个单词/话题都记录：
  - 学习日期 (`learnedWordsTimes`)
  - 复习次数 (`learnedWordsReviewCount`) 
  - 最后复习日期 (`learnedWordsLastReview`)

### 2. 艾宾浩斯复习间隔
- **第0次复习**: 学习1天后
- **第1次复习**: 上次复习3天后  
- **第2次复习**: 上次复习7天后
- **第3次复习**: 上次复习15天后
- **第4次+复习**: 上次复习30天后

### 3. 复习内容选择
- 每次复习选择：**5个单词 + 1个话题**
- 优先选择到期需要复习的内容
- 如果不够5个，补充一些旧内容

### 4. 复习流程
1. 用户点击"开始复习"
2. 系统按艾宾浩斯曲线选择内容
3. AI提示：用这5个单词结合1个话题编故事
4. 用户回答后，自动标记复习完成
5. 更新复习次数和时间

## ? 数据结构（简单不复杂）

```
单词: ["word1", "word2", "word3"]
学习时间: ["2025-07-01", "2025-07-02", "2025-07-03"]  
复习次数: [0, 1, 2]
最后复习: ["", "2025-07-02", "2025-07-04"]
```

## ? 复习逻辑示例

假设今天是2025-07-05：
- word1: 学习4天了，复习次数0 → 需要复习（1天间隔已过）
- word2: 上次复习3天了，复习次数1 → 需要复习（3天间隔已过）  
- word3: 昨天复习过，复习次数2 → 不需要复习（7天间隔未到）

## ? 优点
- **简单**: 只用基础数据类型，不易出bug
- **有效**: 遵循记忆规律，科学复习
- **自动**: 无需手动管理，系统自动计算
- **实用**: 每次5词1题，适合短时间复习
